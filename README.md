# The Gathering
#### Magic the Gathering card finder, using the Magic API.

To use this card selector:
- Enter a card name
- If you don't know this game enter one of the example cards below:
  - Braid of Fire
  - Master of the Wild Hunt
  - Karrthus, Tyrant of Jund
  - Deafening Silence
